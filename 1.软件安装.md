# 配置Node.js
* 注意安装时要添加path
1. npm i -g nrm
2. npm ls
3. npm use taobao
4. npm i -g http-server

# 配置yarn
* 注意安装时要添加path
1. 去官网下载yarn
2. yarn global add yrm
3. yrm ls
4. yrm use taobao
5. yarn global add http-server
